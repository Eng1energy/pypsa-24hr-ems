# PyPSA 24-Hour Energy Management System (EMS)

**Real-time optimal dispatch** for a renewable-powered home — **solar + wind + battery + EV**.

Built live on November 11, 2025 — from scratch.

## Features
- 24-hour simulation (hourly resolution)
- 10 kW solar + 6 kW wind
- 7 kW / 42 kWh lithium battery
- Electric vehicle charging (11 kW, 6–9 AM)
- Carbon-aware + cost-optimal dispatch
- Full interactive plot
